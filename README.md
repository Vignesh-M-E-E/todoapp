# Frontend - Angular Todo App

Angular frontend for the Todo application.

## Running the Application

1. Make sure you have Node.js 18+ and npm installed
2. Install Angular CLI globally: `npm install -g @angular/cli`
3. Navigate to the frontend directory
4. Install dependencies: `npm install`
5. Run: `ng serve`
6. Open `http://localhost:4200` in your browser

## Components

- **RegisterComponent** - User registration page
- **LoginComponent** - User login page
- **TodoComponent** - Todo creation page

## Services

- **AuthService** - Handles registration and login API calls
- **TodoService** - Handles todo creation API calls
